# code-tell-you-javascript

通过在线的jupyter notebook运行一些典型的JavaScript代码

> 使用了binder平台的docker模式，启动需要一定时间
> [binder文档](https://mybinder.readthedocs.io/en/latest/)

## Usage

点击文档顶部的binder图标，在线执行代码
